This folder is the designated location for datasets used during the CariSurg Healthcare AI Training Program. The Week 0 dataset belongs here.

Important: Do not commit any patient‑identifiable information to this repository. Before adding files, ensure data are de‑identified, synthetic, or otherwise authorised for public storage.

When adding a dataset, include a short README entry describing:

Source and licence

Contents and format

Any preprocessing performed

Files in this folder are for analysis and demonstration only. If sensitive data are required for future work, store them outside the public repository and document access procedures here instead.
